# 信息检索技术课程大作业

## 大作业要求

从零开始搭建一个自己的企业智能客服系统

- [x] Step 1 数据获取与处理：
  - 爬虫和文本处理知识：是爬取某个领域的论文或文档，为知识库做准备
- Step 2 索引与检索：
  - 倒排索引和布尔检索：是构建第一版基于关键词的检索器
  - 向量空间模型和TF-IDF：是引入向量化检索
- Step 3 排序与评估：
  - 排序算法：是优化检索结果的排序，Learning to Rank概念
  - 用户评价：是设计实验，对比关键词检索和语义检索的效果差异
- Step 4 前沿与融合：
  - 引入大模型，将自己构建的检索器作为RAG系统的检索模块，与大模型结合，生
  成最终的、有据可查的答案
  - 对比“纯大模型回答”和“RAG系统回答”, 在准确性和实时性上比较差异
